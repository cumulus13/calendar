# Kalendar
Calendar with holiday and day off customize
    
## usage
edit holidays and day off on 'kalendar.ini' config file as you want then just run
```bash:
$ python kalendar.py
```

![example](./screenshot.png)

## requirements (pip)
- rich
- configset
- pydebugger (option for debug only)
